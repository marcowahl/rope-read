The purpose of this page is to support the Rope Read reading style. Browsers confirmed to transform text to rope read: firefox, conkeror, chrome. This is free software. Tip: Page reload empties the text field. Find details and all the code at github.com/marcowahl/rope-read And btw this is not a cloud thing here. The browser does all the work. No data gets stored by the maintainer/developer of this page. Have fun!
